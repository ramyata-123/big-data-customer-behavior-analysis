# Customer Purchase Behavior Analysis using Amazon Purchase and Survey Data

## Project Overview
This project analyses Amazon purchase and survey datasets to understand customer purchasing behaviour and generate business insights.

## Technologies Used
- Python
- PySpark
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Project Structure

```
Customer_Behavior_Analysis.ipynb
Big_Data_Project_Report.pdf
data/
images/
README.md
requirements.txt
```

## Key Analyses
- Purchase Trends
- Product Performance
- Seasonal Revenue Analysis
- Customer Segmentation using RFM
- K-Means Clustering
- Fraud Detection
- Business Recommendations
